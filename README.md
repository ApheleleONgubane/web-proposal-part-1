-Project Overview

This project is the development of a website for Charcoal Aluminium Windows and Doors. The website is designed to promote the business by showcasing its products and services, improving customer communication, and increasing online visibility. It provides important information about the company and allows customers to make enquiries easily.

Website Goals and Objectives:
-To create an online presence for the business
-To advertise aluminium windows and doors products
-To provide clear information about the company
-To allow customers to contact or request quotations easily
-To improve customer engagement and accessibility
-To attract new clients and grow the business

Key Features and Functionality
-Home page with business introduction
-About Us page with company background, vision, and mission
-Products/Services page showing aluminium windows and doors
-Enquiry form for customer requests
-navigation menu for easy browsing
-Contact details and working hours
-Images of products for better presentation

 Part 2 Updates

 CSS Styling Enhancements

The website was improved by creating an external CSS stylesheet (style.css) and linking it to all HTML pages. The styling provides a consistent appearance across the website and improves user experience.

 Improvements Made

* Added external CSS stylesheet.
* Linked all HTML pages to style.css.
* Added a professional colour palette based on charcoal and aluminium colours.
* Styled the navigation menu.
* Added hover effects to navigation links and buttons.
* Styled forms and input fields.
* Styled images with borders, rounded corners, and shadows.
* Added content containers using div classes.
* Styled the footer section.
* Added responsive layouts using Flexbox.
